---

`@nest-boot/hash` 模块使用 bcrypt 或类似算法提供安全的密码哈希和验证。

## 安装

```bash
pnpm add @nest-boot/hash
```

## 模块注册

```typescript
import { HashModule } from "@nest-boot/hash";

@Global()
@Module({
  imports: [HashModule],
})
export class CommonModule {}
```

## 用法

```typescript
import { HashService } from "@nest-boot/hash";
import { Injectable } from "@nestjs/common";

@Injectable()
export class AuthService {
  constructor(private readonly hashService: HashService) {}

  async hashPassword(password: string): Promise<string> {
    return this.hashService.hash(password);
  }

  async verifyPassword(password: string, hash: string): Promise<boolean> {
    return this.hashService.verify(password, hash);
  }
}
```

## MikroORM 集成

使用 `@nest-boot/mikro-orm-hash` 的 `@HashedProperty()` 装饰器实现自动字段哈希。详见 [MikroORM 教程](/docs/tutorial/mikro-orm#哈希属性)。
