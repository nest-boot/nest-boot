`@nest-boot/graphql` 模块提供由 [Apollo Server](https://www.apollographql.com/docs/apollo-server/) 驱动的 GraphQL API 支持，包括代码优先的模式生成、异常过滤和日志/限流伴生模块。

## 安装

```bash
npm install @nest-boot/graphql @nestjs/graphql @nestjs/apollo @apollo/server graphql
# 或
pnpm add @nest-boot/graphql @nestjs/graphql @nestjs/apollo @apollo/server graphql
```

## 基本用法

```typescript
import { Module } from "@nestjs/common";
import { GraphQLModule } from "@nest-boot/graphql";

@Module({
  imports: [GraphQLModule.forRoot({})],
})
export class AppModule {}
```

## 特性

- **Apollo Server** — 生产就绪的 GraphQL 服务器
- **代码优先** — 从 TypeScript 类型自动生成模式
- **异常过滤** — 全局 GraphQL 错误处理
- **操作日志** — 所有操作的结构化日志
- **请求限流** — Shopify 风格的查询复杂度限流
